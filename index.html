Projects:

Here are some of the projects I’ve worked on.

  • Forecasting furniture sales Code

The aim was to forecast the furniture sales using uni-variate time-series data. Through the course of the project, I explored various time-series analysis techniques, analyzed and formulated them to build the best model for this case.

  • Clustering food item Code

Clustered a database of food items into appropriate segments (veg, non-veg, drinks, etc) using an unsupervised approach, since there are no features in our data, only (item_id, item_name).

I have created cosine similarity matrices using word embeddings like Word2Vec, SpaCy, GloVe. Understood implications and applications of different embeddings

  • Speech emotion recognition Code

Librosa library is used to train a Neural Network MLP Classifier on audio clips with different emotions. The model can determine the emotion of the speech, for eg. sad, happy, calm, normal, etc.

  • Toxic comments classification challenge (Kaggle) Code

Built an LSTM model capable of detecting different types of toxicity in comments like threats, obscenity, insults, and identity-based hate. Utilized GloVe embeddings as it captures context through word-positioning better.
